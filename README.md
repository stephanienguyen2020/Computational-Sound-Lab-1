# WebAudio Synthesizer Lab

This is a browser-based synthesizer built with the Web Audio API.

## Features
- **Polyphonic Synthesizer**: Play multiple notes at once without clipping.
- **Waveform Selection**: Choose between Sine, Square, Sawtooth, and Triangle waves.
- **ADSR Envelope**: Smooth attack and release to prevent audio clicking.
- **Visual Feedback**: Interactive keyboard that lights up when played.
- **Dynamic Background**: The background color subtly shifts based on the pitch of the note played.

## How to Play
- **Row 1**: Z X C V B N M (White keys) | S D G H J (Black keys)
- **Row 2**: Q W E R T Y U (White keys) | 2 3 5 6 7 (Black keys)

## Technologies
- HTML5
- CSS3
- JavaScript (Web Audio API)
